# Exercise 00 — Stakeholder catalog for "Sign up for a haircut" (Task 1)

Дата добавления в каталог: PIM_02

Краткое описание:

Каталог стейкхолдеров для задачи 1 — система онлайн-записи в сеть барбершопов. Файл содержит выбранные характеристики (колонки) и заполненную таблицу с ключевыми стейкхолдерами, известными из условия задачи, а также небольшие пояснения.

Выбранные характеристики каталога (рекомендуемые столбцы):

- Идентификатор — уникальный код стейкхолдера (формат st001NNN)
- Наименование / ФИО — название организации или ФИО физического лица
- Организация / физ. лицо — тип субъекта
- Категория стейкхолдера — (выгодоприобретатель, пользователь, регулятор, проектная команда и т.п.)
- Уровень контактности с системой (слой луковичной диаграммы)
- Роль в проекте / системе — кратко
- Контакт / канал связи — (если применимо)
- Масштабируемость роли — один / несколько / массово
- Ответственность / обязанности — кратко
- Уровень полномочий — что уполномочен решать
- Предыдущий опыт, полезный проекту — если есть
- Дата добавления — PIM_02 (можно заменить реальной датой)
